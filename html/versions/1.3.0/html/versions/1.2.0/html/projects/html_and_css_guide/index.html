coding setup:

first start by making a folder somewere and open it in VScode, if you don't have VScode installed then install it.

i recommend you use this VS code extention so your webpage automaticly updates when saving a file.

after opening the folder make a file and name it "index.html", this way you don't have to give the file in the search bar becouse a browser automaticly runs the index.html file in the folder. this is a self closing tag.

then type "!" and press "Tab", this will autoaticly make the main tructure for an html file.

basic html:

<!DOCTYPE html>

In the html file you will see "<!DOCTYPE html>", this tells the browser that its an html document it will automaticly tell the browser that its the newst version


<html> </html>

Everything between <html> </html> is whats in the html file and everything the browser needs to show the webpage.


<head> </head>

Everything between <head> </head> is informatio about the webpage. Examples of things it includes are the title and a link to the css file.


<title> </title>

Everything between <title> </title> is the title of the web page, this one is called "html & css guide" as you can see on the top of your screen.


<h1> </h1>

Everything between <h1> </h1> is a heading, you have 6 headings, each one smaller then the one before, here are some examples:

<p> </p>

Everything between <p> and </p> is a paragraph, you display text in it, here is an Example:

this is an paragraph

if you want some sample text in VScode then you can type "lorem" and then press tab, this will place this text:

a paragraph doesnt care about how its structured in the code, it doesnt matter if you have too many spaces, blank lines or muliple rows, everything between the <p> and </p> and for example the lorem... above this look like this in the code:
