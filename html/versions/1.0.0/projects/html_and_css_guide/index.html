coding setup:

first start by making a folder somewere and open it in VScode, if you don't have VScode installed then install it.

i recommend you use this VS code extention so your webpage automaticly updates when saving a file.

after opening the folder make a file and name it "index.html", this way you don't have to give the file in the search bar becouse a browser automaticly runs the index.html file in the folder.

then type "!" and press "Tab", this will autoaticly make the main tructure for an html file.


basic html:

<!DOCTYPE html>

in the html file you will see "<!DOCTYPE html>", this tells the browser that its an html document it will automaticly tell the browser that its the newst version

<html> </html>

Everything between <html> </html> is whats in the html file and everything the browser needs to show the webpage.
